# Public demo safety

The public application is intentionally an analyzer, not an autonomous agent.

## Connected capabilities

- local deterministic text processing;
- local embedded classifier inference;
- sanitized D1 audit persistence;
- hashed D1 request-rate tracking.

## Capabilities intentionally absent

- shell or PowerShell execution;
- filesystem access;
- arbitrary outbound fetches;
- browser automation;
- uploads;
- authentication or personal workspaces;
- direct connections to production AI agents.

Do not paste confidential content into the public demo. Redaction is best-effort and cannot recognize every proprietary data format.
